# Perplexity-Client 

Perplexity client is a API-First frontend client, it has all sorts of features that normal perplexity doesn't,
this electron based app also offers a neat user interface with high customizability and a mac-os styled window
title If your looking to use your pro credits that are going to expire anyway, use this!

## Features

- API-based 
- Deep research
- Space (Custom instructions)

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```


Build:
```bash
npm run build:win  # For Windows
npm run build:mac  # For macOS
npm run build:linux # For Linux
```

## Usage

#Basic Usage

-Launch the app via npm start, or download the zip
-Enter your Perplexity API key, or OpenRouter API key in settings
-Use spaces for customized output
-Streaming responses displayed in real-time
-Model selection dropdown (e.g., sonar-pro, sonar-large)
-Search focus options (academic, web, etc.)
-History sidebar for previous conversations

## Code Explanation

-Project total ~2000+ LOC across main/process, renderer, HTML/CSS, utils. Customize API key in settings or env. Run npm start for demo.
